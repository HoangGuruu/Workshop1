Tạo ECS Cluster

Tạo ECS Cluster

  1. Truy cập vào AWS Management Console.
  • Tìm ECS.
  • Chọn Elastic Container Service.
  • Chọn Clusters.
  • Chọn Create Cluster.

  1. Trong giao diện Create Cluster.
  • Cluster name: nhập DEMOGO-ECS
  • Cấu hình Networking:
    • VPC: chọn ECS VPC
    • Subnets: chọn Private Subnet

  1. Trong giao diện Infrastructure.
  • Chọn Amazon EC2 instances
  • Operating system/Architecture: chọn Amazon Linux 2
  • Desired capacity: Minimum: 2, Maximum: 2
  • EC2 instance type: chọn m5.large
  • SSH Key Pair: chọn None

  1. Để có thể giám sát cluster của bạn, trong Monitoring bật Use Container Insights rồi chọn Create.

  1. ECS Cluster đã được tạo.

  1. Để truy cập vào ứng dụng mà chúng ta sẽ triển khai trên cluster, chúng ta cần sửa lại security group của cluster.
  • Truy cập vào EC2
  • Chọn Auto Scaling Groups
  • Chọn Infra-ECS-Cluster-DEMOGO-ECS-XXXX rồi chọn Launch Templates

  1. Trong giao diện Launch templates.
  • Chọn ECSLaunchTemplate_XXXX
  • Chọn Action rồi chọn Modify template (Create new version)

  1. Trong giao diện Network settings.
  • Security groups: chọn ECSInstanceSG
  • Chọn Create template version

  1. Quay lại giao diện Auto Scaling groups.
  • Chọn Infra-ECS-Cluster-DEMOGO-ECS-XXXX
  • Chọn Action rồi chọn Edit

  1. Trong Launch template chọn Version thành 2.

  1. Chọn Update

  1. Chúng ta đã cập nhật thành công Security Group được liên kết với Auto Scaling group của ECS Cluster nhưng nó không được cập nhât với các instance đã được tạo.

  1. Vào Instance refresh rồi chọn Start instance refresh.

  1. Chọn Start instance refresh.

  1. Truy cập vào EC2, chọn Instances. Bạn sẽ thấy các instance đang được tạo lại.

  1. Kiểm tra lại security group và thấy nó đã được thay đổi.
